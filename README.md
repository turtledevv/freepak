# freepak
> flatpak sandbox removal tool

> ![IMPORTANT]
> Be careful, this is a powerful tool! Sandboxing exists for a reason, a SECURITY reason. Removing these measures should only be used on apps you trust!

Tired of Flatpak sandboxing your applications to hell?</br>
Tired of direct paths not working because of sandboxing?</br>
Here's the program for you; freepak!</br>
It's a simple CLI tool that removes most sandboxing from a flatpak application!</br></br>
*Example:*</br>
```bash
freepak app.zen_browser.zen
```
That'll remove all restrictions from Zen, allowing it to access files, devices, sockets, etc..</br>
